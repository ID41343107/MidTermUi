# MidTermUi

A Qt-based Tic-Tac-Toe game with a graphical user interface.

## Features

- 3x3 game grid with large, centered O and X symbols
- Turn-based gameplay (X starts first)
- Click on empty cells to place your mark
- Clean and simple interface

## Building

This project requires Qt5 or Qt6. Build using qmake:

```bash
qmake MidTermUi.pro
make
```

## Playing

- Click on any empty button to place your mark (X or O)
- Players alternate turns automatically
- Once a cell is filled, it cannot be changed
